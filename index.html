How to use

  1. Enter your Impact ID number (found in your Impact Dashboard by clicking on your account name).

  2. Choose a Landing Page from the list. Pages with “*” are recommended.

  3. Pick a language for the Landing Page.

  4. Add parameters like SharedID, SubID 1, SubID 2, and SubID 3 to get more details about the click’s origin from specific sources like articles or CTAs.

The link should have the following structure:

  • Example with no tracking parameters: https://go.expressvpn.com/c/YourPersonalImpactID number/LandingPageID/16063
  • Example with tracking parameters: https://go.expressvpn.com/c/YourPersonalImpactID number/LandingPageID number/16063?subId1=Streaming&subId2=Banner
  • Please check if the link leads to the desired landing page.

Link section explanation:

Base redirect url, Your Impact ID number, Landing Page ID number - Every landing page on Impact has its own, unique identifier number., Advertiser ID - This number won’t change, it is ExpressVPN ID., Additional Parameters (SubID 1, SubID 2, SubID 3 and Shared ID)
